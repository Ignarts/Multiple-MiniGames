### Android
1. Open the project in Unity.
2. Go to `File > Build Settings`.
3. Select `Android` and click `Build and Run`.

### Windows
1. Open the project in Unity.
2. Go to `File > Build Settings`.
3. Select `PC, Mac & Linux Standalone` and ensure that `Target Platform` is set to `Windows`.
4. Click `Build and Run`.
